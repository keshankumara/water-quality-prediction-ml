# Water Quality Prediction (ML)

## About The Project
This project was developed for academic purposes as part of our university coursework. The goal is to predict water potability (whether water is safe to drink) using machine learning techniques.

Because this is an academic project, we intentionally used two models instead of only one. This helped us:
- Compare performance across different approaches
- Understand model behavior and trade-offs
- Build stronger analysis and discussion for our final report

The focus is on learning, experimentation, and clear interpretation of results rather than building a production-ready system.

## Project Objective
- Analyze water quality features from a labeled dataset
- Train and evaluate two ML models for potability prediction
- Compare model outcomes using common evaluation metrics
- Present findings in a simple, explainable way

## Dataset
The dataset used in this project is:
- `ml-model/water_potability.csv`
- Dataset Link: https://www.kaggle.com/datasets/adityakadiwal/water-potability

## Project Structure
- `ml-model/water-quality-prediction.ipynb`: Main notebook for preprocessing, training, and evaluation
- `ml-model/water_potability.csv`: Input dataset
